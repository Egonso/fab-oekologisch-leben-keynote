# FAB Oekologisch Leben Keynote

Delivery bundle for the FAB keynote and workshop package around `oekologisch-leben.org`, Google AI Studio, portfolio building, and the challenge platform prompt.

## Structure
- `research/`
  - website extraction and context packs
- `prompts/`
  - design-document, PRD, and AI Studio platform prompts
- `slides/`
  - speaker-deck source material
- `handout/`
  - participant-facing reference pack
- `scripts/`
  - reproducible extraction helpers
- `assets/`
  - local support files when needed
