# Visual System for the Keynote

## Direction
- industrial minimalism
- institutional calm
- professional clarity
- subtle zen rhythm without cultural cliches

## Color
- Off-White: `#F5F3EE`
- Charcoal: `#1D201C`
- Graphite: `#2A2E29`
- Steel Gray: `#6E756B`
- Accent Green: `#345900`

## Typography Direction
- headline direction: sharp, editorial sans or restrained grotesk
- body direction: clear, readable sans
- monospaced accent only for technical snippets or labels

## Surface Rules
- avoid bubble motifs
- avoid pill-heavy navigation
- use frames, grids, lines, panels, and disciplined whitespace
- prefer split layouts, tables, and editorial sectioning over nested rounded cards

## Motion
- minimal reveal
- short fade/slide transitions
- restrained stagger only where it adds orientation

## Image Rules
- use real screenshots, system diagrams, browser surfaces, and selected documentary imagery
- avoid generic AI robots, glowing brains, neon code tunnels, or stock-futurism
- use diagrams and comparison charts where they teach something concrete
